工具总览

常用工具

命令 功能
vim 文本编辑神器
tmux 分屏神器

vim

  • vim 常用指令：
vim                 # 打开vim
vim [file]          # 打开并编辑文件

tmux

  • tmux 常用快捷键：
Ctrl + b + "        # 创建上下分割的窗口
Ctrl + b + ,        # 创建左右分割的窗口
Ctrl + b + up       # 切换到上窗口
Ctrl + b + down     # 切换到下窗口
Ctrl + b + left     # 切换到左窗口
Ctrl + b + right    # 切换到右窗口
